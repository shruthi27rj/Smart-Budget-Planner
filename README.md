# Smart-Budget-Planner
A Python Tkinter app to manage monthly budgets and visualize expenses

## Overview
This project is a desktop-based budget planner developed using Python and Tkinter. It allows users to enter their monthly income and track expenses across different categories. The application calculates total expenses and remaining balance in real time, and also provides an option to visualize spending patterns using a pie chart.

The goal of this project is to provide a simple and practical tool for basic personal finance management.

---

## Features
- Input and manage monthly income
- Add expenses under predefined categories
- Real-time calculation of total expenses and remaining balance
- Tabular display of entered expenses
- Export expense data to a CSV file
- Visualization of expense distribution using a pie chart

---

## Technologies Used
- Python
- Tkinter (for graphical user interface)
- Pandas (for data handling)
- Matplotlib (for visualization)
